# Microservicio Reporte

Gestión de reportes del sistema con persistencia en MongoDB.

## Requisitos

- Java 25+
- MongoDB 4.0+
- Gradle 9.3.0+

## Levantar el servicio

```bash
./gradlew bootRun
```

El servicio estará disponible en `http://localhost:8085`

**OpenAPI (Swagger):** `http://localhost:8085/swagger-ui.html`

**Credenciales predeterminadas:**
- Usuario: `admin`
- Contraseña: `admin123`
